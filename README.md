# UK-Real-Estate-Price-Predictor
This project uses Multiple Linear Regression to estimate the market value of residential properties in Leeds, UK. As a prospective Data Science student, I developed this model to understand how physical features like Square Footage and Number of Bedrooms mathematically influence housing costs.
🚀 Key FeaturesData Processing: Leverages the Pandas library to structure raw housing data into a manageable DataFrame.Feature Engineering: Analyzes two independent variables ($X_1$: Sq_Ft, $X_2$: Bedrooms) to predict one dependent variable ($y$: Price).Machine Learning: Implements the LinearRegression algorithm from Scikit-Learn to find the "Line of Best Fit."Visualization: Uses Matplotlib to generate a scatter plot and trend line, making the data insights easy to digest.📊 Methodology (The "How-To")Data Collection: Created a sample dataset representing typical property values in a Leeds neighborhood.Model Training: Used the .fit() method to allow the AI to "learn" the correlation between house size and price.Inference: Successfully predicted that a 1,100 Sq Ft house with 3 Bedrooms has an estimated value of approximately £309,200.💡 What I Learned (The "Data Science" Insight)Correlation: There is a strong positive correlation ($+0.98$) between size and price.Coefficients: For this specific area, the model suggests that every additional square foot adds roughly £272 to the property value.Outliers: Importance of checking for "dirty data" that might skew the average price.
